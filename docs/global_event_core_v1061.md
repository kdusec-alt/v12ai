# Global Event Core v1061

## Purpose

Five-minute monitoring must not depend only on the active stock name. This layer scans market-wide oil, tariff and scheduled PMI evidence independently, then reuses the existing V12 routes:

`Global Event Core -> Event Watch/Lifecycle -> Macro/Policy -> Quantum -> AI narrative -> price reality veto`

## Safety boundaries

- Does not write Prediction, Audit, DNA, Learning or V13 Research memory.
- Does not directly overwrite Direction, T0, T1, High, Low or Confidence.
- Price reality and existing overseas/cross-market confirmation remain the final veto.
- Offline tests do not make network calls or inject live macro overrides.
