# UI acceptance points

1. Recent yellow/red event captions must render bright white with full opacity inside expanders.
2. Oil, Taiwan tariff and PMI events must appear as Global Event Core evidence even if the active ticker is SKHY or another unrelated symbol.
3. Oil and tariff titles include the transmission chain rather than only the headline.
4. Same-severity oil quote updates do not create a new alert every five minutes; severity escalation does.
